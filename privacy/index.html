隱私權政策

生效日期：2026 年 2 月 13 日

歡迎使用 LinkTrove（以下稱「本服務」）。我們重視您的隱私，並致力於保護您的個人資料。本隱私權政策說明本服務蒐集、使用、儲存與揭露資料之方式。

一、適用範圍

本隱私權政策適用於您使用 LinkTrove Chrome 擴充功能及其相關功能時所涉及之資料處理行為。

二、我們蒐集與處理的資料類型

1. 驗證資訊

  • Google OAuth 授權權杖（用於 Google Drive 同步）
  • 使用者自行輸入之 GitHub Token（用於 GitHub Gist 分享）

2. 網頁記錄

  • 使用者目前開啟分頁之網址（URL）、頁面標題、分頁群組資訊（tab groups）

3. 網站內容

  • 使用者操作時擷取之頁面中繼資料（例如頁面標題、描述、網站名稱、作者、連結等）

4. 使用者自行建立或匯入之內容

  • 書籤卡片、分類、群組、備註、模板欄位等管理資料

三、資料使用目的

我們處理上述資料僅用於以下目的：

  1. 提供分頁與書籤之收藏、分類、搜尋與整理功能
  2. 提供使用者選擇啟用之雲端同步備份功能
  3. 提供使用者主動觸發之分享功能（例如發佈至 GitHub Gist）
  4. 改善功能穩定性與錯誤排除（不含廣告追蹤用途）

四、資料儲存與第三方服務

1. 本地儲存

  • 多數資料儲存於使用者瀏覽器本機（例如 IndexedDB、chrome.storage.local）

2. Google Drive（選用）

  • 僅在使用者主動啟用同步時，資料會儲存至使用者自己的 Google Drive appDataFolder 私有空間

3. GitHub Gist（選用）

  • 僅在使用者主動執行分享時，選定內容會送至 GitHub Gist

五、我們不會做的事

  1. 不會販售您的個人資料
  2. 不會在未經您操作的情況下任意上傳您的內容到第三方
  3. 不會下載或執行遠端程式碼（所有可執行程式碼皆隨擴充功能封裝）

六、資料保存期間

我們會在達成前述目的所需期間內保存資料。您可隨時刪除本地資料、停用同步、移除擴充功能，或撤銷第三方授權。

七、您的權利

依適用法令，您得行使查詢、閱覽、更正、刪除、停止處理等權利。您亦可透過下列方式自行管理資料：

  1. 刪除本地書籤與設定
  2. 解除 Google Drive 同步授權
  3. 刪除或更換 GitHub Token

八、資料安全

我們採取合理之技術與管理措施保護資料安全，以避免未經授權存取、竄改、揭露或毀損。

九、未成年人隱私

本服務並非專為未成年人設計。若您為法定代理人並認為未成年人提供了不當資料，請與我們聯絡。

十、隱私權政策之修訂

我們可能因功能更新或法令要求調整本政策。修訂後版本將公布於本頁面，並載明生效日期。
